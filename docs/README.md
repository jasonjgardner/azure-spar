# Azure Spar Documentation

RenderDragon material binary format reference, originally from the [lazurite](https://github.com/veka0/lazurite) project. The on-disk `.material.bin` format is identical between lazurite and azure-spar.

## Contents

| Document | Description |
|----------|-------------|
| [Material Format](material.md) | `.material.bin` binary structure: buffers, uniforms, passes, variants, shaders |
| [Shader Platforms](platforms.md) | Supported RenderDragon shader platforms (Direct3D, ESSL, GLSL, Metal, Vulkan) |
| [Supported Versions](supported_versions.md) | Material format versions (22-25) and corresponding Minecraft releases |
